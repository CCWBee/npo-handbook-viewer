# NPO Handbook Viewer

A browser reader for the Handbook for the Countering of Terrorist Financing for Prescribed Non-Profit Organisations. The handbook text lives in `policy.md` and is rendered client-side, so the whole thing is a single static page you can open anywhere.

An unofficial reading aid, put together to make the guidance easier to navigate. It does not replace the official published handbook, which remains the authority.

## Use

Open `index.html` in a browser.
