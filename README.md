# NPO Handbook Viewer

The Handbook for the Countering of Terrorist Financing for Prescribed Non-Profit Organisations is published as a PDF. This turns it into a single, searchable, machine-readable web page: the same text, but navigable, linkable and interactive rather than something you scroll and squint at.

The handbook content lives in `policy.md` and is rendered client-side, so it opens anywhere with no server.

## Use

Open `index.html` in a browser.

## Standing

An old personal prototype, built in my own time and long since moved on from. It is not a work product, and it is not affiliated with, endorsed by or representative of any employer or of the body that publishes the handbook. It does not replace the official published handbook, which remains the authority.
